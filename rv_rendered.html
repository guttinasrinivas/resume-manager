ECG Sensor For Cloud Analytics
Product
CardioNebula
Tools
Embedded C, Embedded C++, KiCAD, nRF24L, Logic Analyzer, Oscilloscope, Battery Analyzer
Platforms
MSP430, AVR, ARM
Summary
The product acquires data from the ECG probes and uploads to the Cloud for analytics. The Cloud Analytics attempts to detect anomalies in the input pattern and alert the doctor if required.
Contributions
Design the hardware and ECG probe interfaces. Develop prototypes and verify the functionality. Write firmware on bare-metal and Embedded Linux Drivers Lay the PCB out and generate BOM.
